What is the difference between Margin, Border, and Padding?

The grey background shows the margin from the orange border to the outside of this div element

My content is padded from this text to the orange border

The above diagram is a working example, feel free to inspect its properties

The 'box model' is a box (shown above) that wraps around every html element, consisting of margins, border, padding and the actual content.

Margins create spacing between your elements border and its parent, they are measured using common values and can be poitive or negative. Margins are a common styling tool to create space around your elements, often to prevent them from being right up against other content. Margins can be adjusted on all sides (top, right, bottom, left). To create you own margins use the following styling in your CSS stylesheet: margin: top right bottom left; | Example margin: 10px 20px 10px 20px;

Read more on margins here

Borders represent the outside of your element. Borders are invisibile by default but can be styled using different border styles, colors and thicknesses. To create your own border use the following styling in your CSS stylesheet: border: style thicknesses color; | Example border: dotted 3px black; (see below examples)

Read more on borders here

Padding is the spacing from the edge of your element to your elements content. Padding can be adjusted on all sides (top, right, bottom, left) and are a common styling tool to create spacing between your container and its content. To create your own padding use the following styling in your CSS stylesheet: padding: top right bottom left; | Example padding: 10px 15px 10px 15px;
